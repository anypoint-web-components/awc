Anypoint web components - Base components

A set of a base web components that renders Material Design and Anypoint styles.

Base components

anypoint-item
A styles list item.
anypoint-listbox
A pre-configured element that renders list items.
anypoint-checkbox
A styled checkbox.
anypoint-switch
A styled switch button.
anypoint-button
A styled base button.
anypoint-input
Anypoint input elements.
anypoint-selector
The selector for lists.
anypoint-dropdown
An element that renders a dropdown content.
anypoint-tabs
Styles tabs.
anypoint-menu-button
An icon-button that renders a drop down menu.
anypoint-dropdown-menu
A drop down input element.
anypoint-autocomplete
An element that works with an input element to render a dropdown list of suggestions.
anypoint-combobox
An input element with a dropdown list.
anypoint-input-combobox
An input element with a dropdown list.
anypoint-radio-button
A styles radio button and a radio group.
anypoint-collapse
An element with collapsable content.
anypoint-dialog
A styled dialog element.
anypoint-chip
A styled chip (pill) element.
anypoint-chip-input
An input for chips (pills).
A styled chip (pill) element.
scroll-threshold
A utility element that listens for `scroll` events from a scrollable region and dispatches events to indicate when the scroller has reached a pre-defined limit.
scroll-threshold with a UI region
As above but declaring a scrollable UI region.
scroll-threshold decoupled
As above with a decoupled scrollable UI region.
date-time
Date label from a timestamp or a date string.
bottom-sheet
A dialog that renders content at the bottom of the screen.
star-rating
An element to render a number of stars as a rating.

Mixins

validator-mixin
A mixin to implement a validation.
validatable-mixin
A mixin to be applied to an element that can be validated.
menu-mixin
A mixin to be applied a menu behavior.
resizable-mixin
Coordinate the flow of resize events between "resizers".
fit-mixin
Fits an element in another element using `max-height` and `max-width`, and optionally centers it in the window or another element.
overlay-mixin
An element that can be hidden or shown, and displays on top of other content.
scroll-target-mixin
An element to respond to scroll events from a designated scroll target.
scroll-target-mixin - scroll region
As above but declaring a scrollable UI region.
